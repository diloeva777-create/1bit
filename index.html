Первый БИТ: системные улучшения качества сервиса

Документ подготовлен на основе открытых отзывов клиентов (2020–2025) и предлагает конкретные шаги по внедрению единого стандарта обслуживания, SLA и автоматизации контроля.

🏢 Компания: масштаб и контекст

🌍 9 стран 👥 9000+ сотрудников 🏙️ 80+ городов РФ

🏭 40+ отраслей 🎯 Конкурент: 1С-Рарус ⚙️ Франчайзи 1С

✓ Рыночная позиция: лидер по автоматизации в сегменте 1С, но клиентский сервис неравномерен по регионам.

✓ Стратегическая цель: превратить качество обслуживания в конкурентное преимущество.

💡 Главный инсайт из клиентских отзывов (2018–2025):
• 80% позитива связано с конкретными сотрудниками (Владимир, Наталия Кретова)
• Негатив — системные сбои: игнорирование, срыв сроков, навязывание услуг
→ сервис держится на "звёздах", но отсутствует единый стандарт.

⚠️ Слабые стороны: сервис = везение, нет CSAT-опросов, слабая эскалация.

📅 План внедрения системы качества (90 дней)

🔹 МЕСЯЦ 1
Диагностика + быстрые результаты
• Аудит 20 последних жалоб
• Запуск пост-сервисного опроса (SMS/робот)
• Утверждение SLA для 1-й линии поддержки
🔹 МЕСЯЦ 2
Стандарт и пилот
• Единый стандарт качества (скрипты, эскалация)
• Пилот по скорингу в 3-х офисах
• База знаний + наставничество (лидеры обучают)
🔹 МЕСЯЦ 3
Масштабирование и автоматизация
• Федеральный дашборд качества (рейтинг офисов)
• ТЗ на AI-помощника оператора
• Презентация первых KPI руководству
🎯 Ожидаемые бизнес-результаты через квартал: рост CSAT с ~3,8 до 4,4, сокращение времени реакции на 40%, прозрачный рейтинг каждого офиса.
